# Restaurant Menu Management APP

## Project Description

This project is a full stack application built with:

* ASP.NET Core Web API
* Entity Framework Core
* SQL Server
* React
* xUnit
* NSubstitute

The application allows restaurant staff to manage categories and menu items.

## Features

### Categories

* View categories
* Create categories
* Update categories
* Delete categories

### Menu Items

* View menu items
* Create menu items
* Update menu items
* Delete menu items

## Backend Technologies

* ASP.NET Core Web API
* Entity Framework Core
* SQL Server
* Repository Pattern
* Generic Repository
* Dependency Injection
* DTOs

## Frontend Technologies

* React
* React Router

## Testing

* xUnit
* NSubstitute

## Running the Backend

1. Open the solution in Visual Studio.
2. Set RestaurantMenu.API as Startup Project.
3. Run the application.
4. Swagger will open automatically.

## Running the Frontend

1. Open the frontend folder.
2. Run:

npm install

3. Run:

npm run dev

4. Open the URL displayed by Vite.

## Running Tests

Run all tests from Visual Studio Test Explorer.
